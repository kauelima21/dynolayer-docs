Introdução

No DynoLayer, é possível remover dados de duas formas: fazendo um delete ou um destroy.

Abaixo, haverão exemplos de como implementar ambos os casos.

Removendo registros

Ambos os casos resultam na remoção de um item da tabela. Ao usar o destroy, basta indicar a chave primária e ação será executada.
No caso do delete, deve-se primeiro criar uma instância da model com os dados do registro.
