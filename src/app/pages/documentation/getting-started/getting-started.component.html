Criando uma model

Para criar uma model utilizando o DynoLayer, é preciso criar a classe que representará a entidade e fazer com que estenda a classe da lib.
Como exemplo, segue a implementação de uma model User.

Abaixo, o que cada parâmetro representa:
  • entity: é o nome da tabela.
  • required_fields: campos obrigatórios ao salvar um novo registro.
  • fillable: campos que serão preenchidos quando utilizado o método create.
  • timestamps: indica se os registros terão created_at e updated_at. Por padrão, seu valor é True.
